# 🍽️ Restaurant Data Analysis Project

This project focuses on performing Exploratory Data Analysis (EDA) on a restaurant dataset to extract meaningful insights about cuisines, ratings, price ranges, and restaurant distribution.

## 📌 Project Overview
The dataset contains information about 9551 restaurants with 21 features including location, cuisines, ratings, price range, and votes. :contentReference[oaicite:0]{index=0}  
The goal of this project is to analyze patterns and trends in the data using Python.

- Seaborn
- Folium (for maps)
